# A2+ English Maintenance Map

Portable, offline-first English practice handout. Development requires Node.js 24 or newer.

## Commands

```powershell
npm install
npm run test
npm run build
```

The delivery file is `dist/a2-english-maintenance-map.html`. Open that file directly in a browser; it does not need a server, login, or internet connection. The `dist` folder contains no supporting JavaScript, CSS, image, or data files.

## Progress and privacy

Checked/completed task state is saved only in the current browser on the current device under a handout-versioned local key. If browser storage is unavailable, the handout still works for the current tab. Use **Reset this handout** to clear its saved progress.

Unfinished typed answers and personal free-response writing are never saved between sessions. Personal prompts can be marked complete but are not automatically scored.
